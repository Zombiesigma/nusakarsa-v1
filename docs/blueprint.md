# **App Name**: NusaKarsa

## Core Features:

- Secure User Authentication: Allows users to create accounts, log in, and securely manage their personal ebooks and content.
- Personal Ebook Library: Enables users to store, view, and organize their personal collection of ebooks within the application.
- Interactive Ebook Reader: Provides an engaging reading experience with smooth chapter navigation, adjustable display settings, and responsiveness.
- Feature-Rich Ebook Editor: Offers a robust environment for users to write and edit ebook content with rich text formatting, and content organization tools.
- AI Writing Assistant Tool: Integrates an AI tool within the editor to provide writing suggestions, grammar checks, and creative prompts, aiding authors in content creation.
- Responsive User Interface: Ensures the application is fully functional, visually consistent, and aesthetically pleasing across all device sizes and orientations.

## Style Guidelines:

- Primary color: A vibrant purple (#7E47EB) to evoke creativity and a modern, energetic feel for digital content.
- Background color: A subtle and desaturated lavender (#F5F1FA) to provide a clean and focused backdrop that allows content to stand out.
- Accent color: A dynamic orange (#F29E1A) used for interactive elements, highlights, and calls to action, complementing the primary purple and adding warmth and energy.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, slightly techy aesthetic, reinforcing the energetic and digital nature of NusaKarsa.
- Body and UI font: 'Inter' (sans-serif) for its excellent legibility across various screen sizes, ensuring comfortable reading and clear interface navigation.
- Utilize a consistent set of modern, minimalist line-art icons that provide clarity without distracting from the primary content.
- Adopt a responsive, content-first layout emphasizing clear hierarchy and readability for the ebook reader, and intuitive organization for the editor.
- Incorporate subtle and fluid animations for transitions, navigation, and feedback, enhancing the overall user experience without being intrusive.
